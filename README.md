# 殴性格診断

友達グループ内で「あなたは○○タイプ!」と診断される、身内ノリ全開の性格診断Webアプリ。

8問の4択質問に答えると、5人のメンバー(島だけさん / ミナノさん / ピリンチさん / ぐらにうさん / 面真さん)の中から最も近いタイプが診断されます。

## 主な機能

- 8問4択のクイズ形式(進行バー・アニメーション付き)
- 回答結果からスコアを集計し、最も近いメンバータイプを診断
- 結果カードをガチャ・カオス感のあるデザインで表示
- 「結果をコピー」ボタンで診断結果テキストをクリップボードにコピー
- 「画像をコピー」ボタンで結果カードを画像化してクリップボードにコピー(Discordにそのまま貼り付け可能)
  - 画像コピー非対応ブラウザでは「画像をダウンロード」ボタンにフォールバック

詳しい仕様は [spec.md](spec.md) を参照してください。

## 技術構成

- HTML / CSS / JavaScript の単一ファイル構成(ビルド不要・依存パッケージなし)
- 画像化には [html2canvas](https://html2canvas.hertzen.com/)(CDN経由)を使用

## ローカルでの確認方法

`index.html` をブラウザで直接開くか、簡易サーバーを立てて確認できます。

```bash
npx serve .
```

## デプロイ(Vercel)

リポジトリ直下に `index.html` があるため、ビルド設定なしでそのまま静的サイトとして公開できます。

1. [Vercel](https://vercel.com/) にこのリポジトリをインポート
2. Framework Preset は "Other"(自動検出)のままでOK
3. Deploy を実行するとURLが発行される

または Vercel CLI で直接デプロイ:

```bash
vercel
```
